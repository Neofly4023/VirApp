Help

1. Comment se connecter à l'application ?

1- Saisir le nom d'utilisateur

2- Saisir le mot de passe

exemple

Nom d'utilisateur : mon_nom_d_utilisateur

Mot de passe : mon_mot_de_passe

3- Cliquer sur le bouton "Se connecter"

2. Comment modifier mon compte ?

1- Cliquer sur le nom d'utilisateur

2- Cliquer sur mon compte

3- Appliquez les changement que vous voulez que ça soit ( Nom d'utilisateur ou de Mot de passe )

4- Clique sur enregistrer les modifications

3. Comment creer un compte simple utilisateur ?

1- Cliquer sur le nom d'utilisateur

2- Cliquer sur creer un compte simple utilisateur

3- Remplir les champs Nom, Prenom, Nom d'utilisateur, Mot de passe

4- Confirmer le mot de passe

5- Cliquer sur créer

4. Comment Supprimer un compte simple utilisateur ?

1- Cliquer sur le nom d'utilisateur

2- Cliquer sur supprimer un compte simple utilisateur

3- Sélectionner le simple utilisateur que vous voulez supprimer

4- Cliquer sur supprimer

5. Comment modifier un compte simple utilisateur ?

1- Cliquer sur le nom d'utilisateur

2- Cliquer sur modifier un compte simple utilisateur

3- Sélectionner le simple utilisateur que vous voulez modifier

4- Modifier selon votre choix le nom d'utilisateur ou le mot de passe

5- Cliquer sur enregistrer les modifications

6. Comment ajouter une demande de prime ?

1- Cliquer sur demande de prime se trouvant sur le menu

2- Sélectionner le type de demande selon votre choix (Mariage, Nouveau-né, Don, Circoncision, Retraite, d&aecute;cès parents, Ramdan, Entrée sociale, d&aecute;cès employe)

3- Sélectionner la date de l'évenement en cliquant sur le calendrier

4- Remplir les champs Nom et Prenom du demandeur

5- Sélectionner la date de la demande en cliquant sur le calendrier

6- Cliquer sur le bouton Ajouter

Remarques

Remarque 1:

Dans le cas ou le type de la demande est un d&aecute;cès

une nouvelle fenêtre s'ouvrira

1- Sélectionner le type de décès: (a- Employée: dans le cas où c'est l'employe en question qui est mort) ou (b- Parent: dans le cas où c'est un parent à l'employe qui est mort)

2-a- Sélectionner la date de l'evenement sur le calendrier

3-a- Remplir les champs Nom et Prenom du demandeur

4-a- Choisir la Situation du demandeur (Mme pour une dame, Mlle pour une demoiselle, Mrs pour un monsieur)

5-a- Remplir les champs Nom et Prenom du fonctionnaire décédé

6-a- Sélectionner le lien de parenté (Pere, Mere, Fils, Frere , Soeur, Fille, Mari, Femme)

7-a- Selectionne la date de demande sur le calendrier

8-a- Cliquer sur bouton Ajouter

2-b- Sélectionner la date de l'evenement sur le calendrier

3-b- Remplir les champs Nom et prenom du demandeur

4-b- Sélectionner la date de la demande sur le calendrier

5-b- Cliquer sur le bouton Ajouter

Remarque 2:

Dans le cas ou le type de la demande est un don

une nouvelle fenêtre s'ouvrira

1- Sélectionner la date de la demande sur l'agenda

2- Remplir les champs Nom et prenom du fonctionnaire

3- Remplir la valeur du montant du don

4- Sélectionner la date de la demande sur l'agenda

5- Cliquer sur ajouter

8. Comment gérer le traitement des demandes( sélectionner la décision ainsi que son motif) ?

4- Cliquer sur new PV

17. Comment importer une table excel ?

1- Cliquer sur Importation Excel se trouvant sur le menu

2- Cliquer sur Parcourir et Sélectionner un fichier excel

3- Choisir le numèro de table en remplissant ce champ ou faire pivoter en cliquant sur table précédent ou table prochaine

19. Comment modifier le montant d'une prime ?

2- Cliquer sur primes

3- Séléctionnez le type de prime à modifier

4- Entrez le nouveau montant et cliquez sur modifier

20. Comment modifier les données d'un fonctionnaire ?

1- Cliquer sur modifier les données se trouvant sur le menu

2- Cliquer sur fonctionnaire

3- Séléctionner le type de départ

4- Remplir les champs matricule du fonctionnaire et motif ainsi que la date

5- Cliquer sur modifier

21. Comment ajouter un fonctionnaire ?

1- Cliquer sur ajouter des données se trouvant sur le menu

2- Cliquer sur Fonctionnaires

3- Remplir les champs Matricule, Nom et Prenom

4- Sélectionner la date de rectrutement sur l'agenda

5- Sélectionner la situation familiale (Marié, Célibataire, Veuf, Divorcé)

6- Cocher sur Oui dans le cas où l'employer est retraité, cocher sur Non si il ne l'est pas

7- Remplir les champs Numéro de téléphone, Email, Compte CCP et Banque

8- Cliquer sur Ajouter

22. Comment ajouter une prime ?

1- Cliquer sur ajouter des données se trouvant sur le menu

2- Cliquer sur Primes

3- Remplir le champ Montant de la prime

4- Sélectionner le type de la prime

5- Cliquer sur Ajouter

23. Comment ajouter une banque ?

4- Remplir le champ désignation banque

5- Cliquer sur Ajouter
